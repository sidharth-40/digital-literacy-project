Project Overview:
This repository contains my Digital Literacy Portfolio created as part of the CSE0001 course. The project focuses on understanding and applying essential digital skills such as creating a professional online presence, using digital tools, practicing safe internet usage, and communicating effectively online.


Tasks Included:

Task 1 – Infographic
Created a digital literacy infographic using Canva covering key topics such as digital literacy, safe internet practices, useful digital tools, professional online presence, and email etiquette.

Task 2 – Digital Portfolio
Created and updated professional profiles on GitHub, LinkedIn, and HackerRank. A GitHub profile README was also created to showcase basic information and interests.

Task 3 – Platforms
Completed a beginner-level coding problem on HackerRank and created a Digital Literacy Awareness Quiz using Google Forms.

Quiz Link: https://docs.google.com/forms/d/e/1FAIpQLSeg4vhpUVmUePSPvXY6BwGGWBbyamZlyfLU8slSjQlutjJm_w/viewform?usp=dialog

Task 4 – Email Etiquette
Drafted two professional emails:
- Request for assignment deadline extension  
- Application for internship opportunity  
Also created a social media do’s and don’ts checklist.

Task 5 – Cybercrime Awareness
Prepared a case study on phishing and created a prevention checklist with safety tips, including UPI safety and cybercrime reporting information.
